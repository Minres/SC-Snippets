Detailed Description

template<typename T, uint64_t SIZE, int lower_width = 24>
class util::sparse_array< T, SIZE, lower_width >

a simple array which allocates memory in configurable chunks (size of 2^lower_width), used for large sparse arrays. Memory is allocated on demand
